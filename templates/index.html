GA Vocabularies System

Welcome to GA's public vocabulary system. This tool is an instance of VocPrez, a read-only web delivery system for Simple Knowledge Organization System (SKOS)-formulated RDF vocabularies.

Vocabularies

This system will soon publish vocabularies from multiple sources, not just GA itself. These multi-source vocabs will all be presented here to provide a single endpoint for all the vocabularies that GA staff use and that GA recommends others to use.

Current vocabulary sources:

  • Government Geoscience Information Committee (GGIC)

Vocabulary Modelling

Geoscience Australia conceives of vocabularies as being synonymous with SKOS' skos:ConceptScheme class, so each GA vocab is an instance of skos:ConceptScheme. Not all users of SKOS make this conception.

Tools

Profiles

All the content of all vocabs can be accessed individually in web page (HTML) and machine-readable forms. Each list fo vocabs, vocabulary and Concept web page contains a link to the listing of all profiles and formats available for that object. Look for the Alternate Profiles hyperlink.
