About this vocabulary system

This vocabulary system is an open source tool known as VocPrez which is a read-only web delivery system - web pages and API - for Simple Knowledge Organization System (SKOS)-formulated RDF vocabularies. It complies with Content Negotiation by Profile.

Documentation

See the documentation at https://rdflib.dev/VocPrez/.
